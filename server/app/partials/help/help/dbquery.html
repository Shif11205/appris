APPRIS Database was originally developed for the GENCODE/Ensembl gene sets that are updated periodically. APPRIS has now been extended to two other public genome annotations, RefSeq and the UniProt proteomes.

In addition, we have created new gene sets based on the merge of Ensembl+RefSeq+UniProt isoforms for each gene. For the human genome we established a common gene set with the GENCODE, RefSeq, and UniProt reference sets. The initial cross-reference was generated with the data-mining tool, BioMart, and from there we re-annotated the cross-database relationships manually. For the remaining species we generated the common gene sets with the BioMart tool, although these relationships are not yet manually annotated.

The APPRIS Database needs:

  • Gene identifier or gene name (required),

  • Species name and the reference gene set (optional).

The Gene IDs could be Ensembl (eg. ENSG00000099904) and Entrez gene ID (RefSeq) (eg. 27037).

The Gene Names are from HUGO (eg. RNF215). Using a Gene name, the query is done in all reference gene sets (Ensembl, RefSeq, and UniProt).

Optionally, the user can provide the species name and the reference gene set. If the user does not provide the optional parameters, the gene is searched in all references sets.

The list of gene sets depends on the species but in most of cases the reference gene sets are the following: GENCODE/Ensembl, RefSeq and the UniProt proteomes.
